* link:https://www.mulesoft.com/lp/dl/studio["Download Studio 7, an an Eclipse-based IDE.^", role="button-primary"] 
* xref:api-led-overview["Testing xref as button in new window^", role="button-primary"]
// * link:https://mulesoft.com["Testing link as button in new window^", role="button-primary"]
* Use two browser windows--one to read instructions and one to perform the tasks.
* Ensure that you are able to run test REST requests on a client such as Postman or Advanced REST Client.
